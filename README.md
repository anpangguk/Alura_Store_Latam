# Alura_Store_Latam
Challenge de Alura - Analisis de facturación, ventas, productos y promedios de la tiendas de "Juan" para escoger que tienda cerrar.
Comienzo con la importacion y carga de datos de las tiendas junto a el analisis de facturacion. 
Uso .groupby('Nombre_Tienda') para juntar todas las filas que pertenecen a la misma tienda. 
Aplicando ['Precio'].sum() para sumar el dinero de todas esas ventas.
Usao plt.bar() de Matplotlib para dibujar el gráfico.
